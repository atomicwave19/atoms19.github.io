toast.js
a simple library build for cool js toasts 
            <script src="https://cdn.jsdelivr.net/gh/atoms19/aw-assets/assets/toast.min.js"></script>
just 1.3kb easy to use toasts
simple toast

to make a simple toast just use the Toastclasd

            new Toast("this is a toast")
coloured toasts
the 2nd attribute changes the background colour of the toast 
            new Toast("this is a toast","limegreen")
more

3rd attribute takes in an object and this object can be used to configure tge toast

property function type
textColor can be used to set the text color of the toast string
timing used to set the speed of the toast animation(in seconds) integer or double
closeDelay sets the delay of the closing of the toast(in seconds) integer or double
animation used to change the animation of the toast just enter the name of the animation string
example
new Toast("hi i am a new toast","white",{
                textColor:"black"
                ,closeDelay:4
                ,timing:0.5
                  
            })
